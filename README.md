PROJECT OVERVIEW

The Garments Worker Productivity Analysis project focuses on analyzing real-world production data from a garment manufacturing industry.
This project aims to identify patterns, trends, and factors that influence worker productivity using data analytics and visualization techniques.
The dataset contains detailed information about:
Target productivity
Actual productivity
Working hours
Incentives
Idle time
Over time
Department and team details
This analysis helps understand operational efficiency and performance gaps in the garment sector.
